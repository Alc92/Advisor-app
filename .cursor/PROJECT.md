# Proyecto Cursor

## Cómo usar este repositorio

- Usa `AGENTS.md` raíz como gobernanza global.
- Usa el `AGENTS.md` del módulo afectado antes de proponer cambios.
- Usa `docs/context/` como resumen operativo de las fuentes maestras.
- Usa `.cursor/skills/` para workflows repetibles.
- Mantén prompts y conversación operativa en español.
- Mantén código, clases, carpetas y nombres técnicos en inglés.

## Convenciones de naming

- Commands: `VerbNoun`
- Queries: `Get`, `List` o `Find` + sustantivo
- Ports: capacidad + `Port` cuando aplique
- Repositories: agregado + `Repository`
- Controllers: sustantivo + `Controller`
- DTOs: solo cuando aclaren el borde

## Regla principal

No reabras decisiones cerradas. Implementa con la mínima ceremonia posible.
